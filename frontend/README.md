# MSE2FSD Frontend

React + Vite frontend for the existing expense tracker backend.

## Local setup

```bash
cd frontend
npm install
npm run dev
```

API URL is separated via env config.
Use `.env`:

```env
VITE_API_URL=https://mse2fsd.onrender.com
```

## Render deployment (Static Site)

1. Push this repository to GitHub.
2. In Render, click `New +` -> `Static Site`.
3. Select this repo and set:
   - Root Directory: `frontend`
   - Build Command: `npm install && npm run build`
   - Publish Directory: `dist`
4. Add environment variable:
   - `VITE_API_URL=https://mse2fsd.onrender.com`
5. Deploy.

After deploy, your frontend URL will be generated by Render.
